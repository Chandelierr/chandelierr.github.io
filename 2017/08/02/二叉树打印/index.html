二叉树打印

2017-08-02 • 标签: 算法

案例1—按层打印二叉树

  • last: 表示正在打印的当前行的最右结点
  • nlast: 表示 下一行的最右结点

思路：利用这两个结点和一个队列，将1放入队列中，另last = 1；然后遍历1的子结点，将2、3放入队列中，这时队列是3 2,遍历的时候令nlast = 2，nlast = 3；遍历完成之后，看last == 1，换行，并另last = nlast；取出队列中的2，将2的子结点放入队列中，并nlast = 4，这时队列4 3,发现last != 2,继续取3，将3的子结点加入队列中，这时队列中6 5 4,并nlast = 6，发现3 == last,换行；取4,放子结点，没有，4 != last,继续；取5，将子结点放入队列，队列这时8 7 6,并令nlast == 8，5 !=last，继续；取6，无子结点，6 == last,换行；取7，8，打印。
